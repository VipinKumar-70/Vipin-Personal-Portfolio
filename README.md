Ensure you have the following installed:

- **Git** for version control (https://git-scm.com/)

### Setup

1. **Clone the repository**:

   ```bash
   git clone https://github.com/VipinKumar-70/Vipin-Personal-Portfolio.git
   cd Vipin-Personal-Portfolio
   npm install
   npm run dev
   ```
